# Create WIBU NFT

Run
```
npm i
```

### Prepare .env file

### Deploy
```
node ./scripts/deploy.js
```

### Mint: 
Prepare and upload your own waifu image and attributes file (.json). Change receiver's address and URI to json file in `mint.js`
```
node ./scripts/mint.js
```
